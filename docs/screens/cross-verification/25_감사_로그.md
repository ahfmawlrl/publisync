## 발견 버그

### BUG-1: FE 타입 누락 — actor_name (Minor)
- **증상:** `AuditLogRecord` 타입에 `actor_name` 필드 없음
- **영향:** BE는 `actor_name`을 반환하고 FE 테이블 컬럼에서 사용하지만 타입에 미정의
- **수정:** `types.ts`에 `actor_name: string | null` 추가

## 쿼리 로직 검증 (3개)

### Q1: list_logs
```python
select(AuditLog).where(organization_id == org_id)
  + optional: action, resource_type, actor_id, start_date, end_date
  .options(selectinload(actor))
  .order_by(created_at.desc())
  .offset/limit
```
- org_id 필터: ✅
- 5개 선택 필터: ✅ (동일 패턴 base+count_base)
- 날짜 필터 ISO 파싱: ✅ (datetime.fromisoformat)
- actor eager load: ✅

### Q2: export_logs
```python
select(AuditLog).where(org_id, created_at >= start, created_at <= end)

```
- 전체 조회 (limit 없음): ✅ (export용)
