## 5. BE 상태 전이 검증

```
UNPROCESSED → {PUBLISHED, HIDDEN, PENDING_DELETE}
PUBLISHED   → {HIDDEN, PENDING_DELETE}
HIDDEN      → {UNPROCESSED, PENDING_DELETE}
PENDING_DELETE → {DELETED, UNPROCESSED}
DELETED     → {} (종료 상태)
```

`_validate_transition()` + `WorkflowStateConflictError` 구현 확인 ✅
