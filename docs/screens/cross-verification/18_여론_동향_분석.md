## 5. 쿼리 로직 검증 (2개)

### Q1: analytics_repository.get_sentiment_trend(org_id, start_date, end_date)
- `JOIN Content ON Comment.content_id == Content.id`: ✅
- `Content.organization_id == org_id` 멀티테넌트 필터: ✅
- `Comment.created_at BETWEEN start_date AND end_date`: ✅
- `GROUP BY date, sentiment` (날짜 × 감성 교차 집계): ✅
- `date_trunc('day', created_at)` → 일별 추이: ✅
- `ORDER BY date ASC`: ✅

### Q2: analytics_service.get_sentiment_trend() — 키워드 빈도 계산
- 댓글 텍스트에서 `re.findall(r"[가-힣a-zA-Z]{2,}", text)` 키워드 추출: ✅
- Counter 상위 500개 제한 (`most_common(500)`): ✅
- 부정 급증 알림: negative 비율 > 0.2 → alert 생성: ✅
- 신뢰도(confidence) 계산: 댓글 수 기반 정규화: ✅
