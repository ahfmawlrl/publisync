## 1. 설계 원칙

### 1.1 벤치마크 분석 요약

| 플랫폼 | 핵심 UI 패턴 | PubliSync 적용 포인트 |
|---|---|---|
| **Hootsuite** | 좌측 사이드바(Streams, Inbox, Planner, Analytics), 중앙 작업 영역 | 좌측 사이드바 기반 네비게이션 채택 |
| **Sprout Social** | 수직 아이콘 네비게이션, Smart Inbox(통합 수신함), 호버 시 라벨 표시 | 아이콘+라벨 네비게이션, 통합 댓글함 설계 |
| **Buffer** | Create→Publish→Analyze→Engage 4단계 흐름, 드래그&드롭 캘린더 | 콘텐츠 생성→승인→게시 워크플로우 중심 설계 |
| **Sprinklr** | 모듈별 탭 구조, 글로벌 검색, 즐겨찾기, 역할별 대시보드 | 글로벌 검색, 역할별 대시보드 적용 |

### 1.2 설계 방향

| 원칙 | 설명 |
|---|---|
| **워크플로우 중심** | 기능 나열이 아닌, 실무자의 업무 흐름(콘텐츠 기획→제작→검수→게시→모니터링→보고)에 따라 메뉴를 구성한다. |
| **AI는 흐름 속에** | AI 기능을 별도 메뉴로 분리하지 않고, 각 업무 단계에 자연스럽게 내장한다. 단, 독립적 AI 기능(리포트 생성, 여론 예측 등)은 해당 섹션에 배치한다. |
| **역할 적응형** | 사용자 역할(시스템 관리자/위탁기관 담당자/수탁업체 관리자/수탁업체 실무자)에 따라 메뉴 항목과 화면 구성을 자동으로 조정한다. |
| **멀티테넌트 전환** | 수탁업체가 관리하는 다수 위탁기관 간 전환이 1클릭으로 가능하도록 상단에 워크스페이스 전환기를 배치한다. |
| **점진적 공개** | Phase별로 추가되는 기능이 기존 메뉴 구조를 깨뜨리지 않고 자연스럽게 확장되도록 설계한다. |
| **반응형** | 데스크톱 full-width 기준으로 설계하되, 태블릿·모바일에서는 사이드바를 축소/오버레이 방식으로 전환한다. |
| **다크 모드** | 라이트/다크 테마 전환을 지원하며, CSS 변수 기반으로 일관된 색상 관리를 한다. 사용자 설정은 브라우저에 저장하여 재방문 시 유지한다. *(v1.2 추가)* |

```
│            │                                                    │
│  [좌측     │  [메인 콘텐츠 영역]                                 │

│            │  ┌──────────────────────────────────────────────┐  │
│  아이콘    │  │  페이지 헤더 (제목 + 액션 버튼)              │  │

│  라벨      │  │                                              │  │
```

### 2.1 상단 바 (Top Bar)

```
┌──────────────────────────────────────────────────────────────────────┐
│ [◎ PubliSync] │ [서울시 홍보담당관실 ▾]   [🔍 검색...]  [🌙][🔔 3][👤] │
└──────────────────────────────────────────────────────────────────────┘
  ↑ 로고 아이콘     ↑ 워크스페이스 전환기    ↑ 글로벌  ↑테마 ↑알림 ↑프로필
  + 텍스트  │구분선  (위탁기관 선택)          검색    전환  센터  메뉴
```

| 영역 | 기능 | 상세 |
|---|---|---|
| **로고** | 홈(대시보드)으로 이동 | SVG 아이콘 + "PubliSync" 텍스트. 클릭 시 대시보드로 이동. 모바일에서는 아이콘만 표시. *(v1.2: 아이콘 추가)* |
| **구분선** | 시각적 분리 | 로고와 워크스페이스 전환기 사이 세로 구분선 (1px). *(v1.2 추가)* |
| **워크스페이스 전환기** | 위탁기관 전환 | 드롭다운으로 관리 중인 위탁기관 목록 표시. 선택 시 해당 기관의 데이터로 전환. 위탁기관 담당자는 자기 기관만 표시. 수탁업체는 "전체 기관" 뷰도 선택 가능. |
| **글로벌 검색** | 통합 검색 | 콘텐츠, 댓글, 미디어, 사용자, 감사 로그 등 전체 엔티티를 키워드로 검색. 검색 결과는 유형별 탭으로 분류 표시. pill 형태(둥근 모서리) 입력 필드. 모바일에서는 숨김. |
| **테마 전환** | 다크 모드 토글 (F-UX) | 라이트/다크 모드 전환 버튼 (🌙/☀️). 선택한 테마는 localStorage에 저장하여 재방문 시 유지. *(v1.2 추가)* |
| **알림 벨** | 알림 센터 (F13) | 미확인 알림 수 배지 표시. 클릭 시 알림 패널(우측 슬라이드아웃) 표시. 알림 유형: 승인 요청, 위험 댓글, 게시 완료/실패, 토큰 만료 등. |
| **프로필** | 사용자 메뉴 | 내 정보, 알림 설정, 로그아웃. 현재 역할 표시. |

### 2.2 좌측 사이드바 (Main Navigation)

기본 상태는 **아이콘+라벨 펼침**(240px). 토글 버튼으로 **아이콘만 표시**(64px)로 축소 가능.

## 6. 다크 모드 지원 *(v1.2 추가)*

### 6.1 개요

PubliSync는 라이트/다크 두 가지 테마를 지원한다. 상단 바의 테마 전환 토글(🌙/☀️)로 즉시 전환되며, 선택한 테마는 브라우저 localStorage에 저장하여 재방문 시 자동 복원한다.

### 6.2 색상 체계

모든 UI 색상은 **CSS 커스텀 프로퍼티(변수)** 로 관리하며, `[data-theme="dark"]` 선택자로 일괄 전환한다. 인라인 스타일에도 변수를 사용하여 테마 전환 시 누락 없이 반영된다.

| 용도 | 라이트 모드 | 다크 모드 |
|---|---|---|
| 배경 (기본) | `#f3f4f6` | `#111827` |
| 배경 (보조) | `#f9fafb` | `#1f2937` |
| 카드/패널 | `#ffffff` | `#1f2937` |
| 테두리 | `#e5e7eb` | `#374151` |
| 텍스트 (기본) | `#111827` | `#f3f4f6` |
| 텍스트 (보조) | `#6b7280` | `#9ca3af` |
| 주요 색상 | `#1a56db` | `#60a5fa` |
| 입력 필드 배경 | `#ffffff` | `#374151` |
| 상태 뱃지 (위험) | `#fef2f2` / `#991b1b` | `#451a1a` / `#f87171` |
| 상태 뱃지 (성공) | `#d1fae5` / `#065f46` | `#0a3622` / `#34d399` |
| 상태 뱃지 (경고) | `#fef3c7` / `#92400e` | `#452a0a` / `#fbbf24` |
| 상태 뱃지 (정보) | `#dbeafe` / `#1e40af` | `#1e3a5f` / `#93c5fd` |

### 6.3 전환 동작

```
사용자 → 상단 바 [🌙] 클릭
         → <html data-theme="dark"> 설정
         → CSS 변수 일괄 전환 (즉시 반영, 리로드 없음)
         → localStorage에 'dark' 저장
         → 아이콘 ☀️로 변경

재방문 시 → 페이지 로드 → localStorage 확인 → 저장된 테마 복원
```

### 6.4 적용 범위

- 상단 바, 사이드바, 카드, 테이블, 폼 요소 등 **모든 UI 표면**에 적용
- 뱃지(상태 라벨)는 다크 모드 전용 색상으로 가독성 보장
- 로그인 페이지 배경 그라데이션도 다크 모드 대응
- 플랫폼 브랜드 색상(YouTube 빨강, Instagram 보라 등 고정 색상)은 테마와 무관하게 유지

```
└── 🔧 플랫폼 전역 설정     ← 시스템 관리자 전용
    ├── API 키 관리
    ├── 텔레그램 봇 설정
    └── 백업·복구
```
